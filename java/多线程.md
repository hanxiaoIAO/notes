# 多线程优势

1. 发挥多核CPU的优势

2. 防止阻塞

   单线程的情况下，如果某个任务耗时较长，会导致整个进程的阻塞。

3. 便于建模

   可以将任务分解，不同的线程执行不同的任务。

# 进程与线程

进程是一个正在运行的程序的实例(an instance of a computer program that is being excuted)。

线程是程序执行过程中的最小单元。
